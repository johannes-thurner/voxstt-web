# VoxSTT web dictation

Browser-based dictation: record, transcribe via the OpenAI API, copy the text.
No backend — your audio and API key go straight from your browser to
`api.openai.com` and nowhere else.

**Live app: https://johannes-thurner.github.io/voxstt-web/**

You bring your own OpenAI API key; it is stored only in your own browser.

This repository contains only the built static bundle. Source lives in the
private VoxSTT repository.
