    Add to Home Screen & Siri

    Install as an app

    In Safari, tap Share → Zum Home-Bildschirm ("Add to Home Screen"). VoxSTT then launches full-screen from its own icon, and your API key and settings stay saved.

    "Hey Siri, VoxSTT Dictate" & the Action Button

    In the Shortcuts app, tap +, add the single action Open URL pointing at this page, and name the shortcut VoxSTT Dictate. Saying “Hey Siri, VoxSTT Dictate” then opens it hands-free; on iPhone 15 Pro+ you can also bind it under Settings → Action Button → Shortcut.

    The microphone prompt

    Launched from the Home Screen, iOS asks for microphone access once per app launch. That is an iOS limitation for standalone web apps, not a bug — grant it and dictate as normal.
